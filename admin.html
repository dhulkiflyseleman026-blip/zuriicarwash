Quick Setup Guide

Follow these steps to go live with real payments.

1️⃣

Add your PawaPay API Token

Go to Payment Gateway → paste your sandbox or production token.

2️⃣

Set your Return URL

The URL PawaPay redirects customers to after payment (your website URL).

3️⃣

Switch to Live Mode

Once PawaPay approves you, toggle from Sandbox → Live in Payment Gateway.

4️⃣

Update Service Prices

Go to Services & Prices to update any prices anytime.
